# End-to-End Sales Data Pipeline & Analytics System

## Overview
This project demonstrates an end-to-end data pipeline for analyzing sales data using SQL and Power BI.

## Tools Used
- SQL / MySQL
- Power BI
- drawDB (database design)
- n8n (automation workflow)

## Features
- Database schema design
- Data ingestion and transformation
- SQL analytics queries
- Interactive Power BI dashboards

## Dashboard Insights
- Total revenue
- Top selling products
- Monthly sales trends
